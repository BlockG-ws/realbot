# realbot
又一个 vibe coding 产物

demo: @BlockG_bot

致力于替代 kmuav2bot，但是很明显还差点距离
# 功能
- /打 \打 这样的指令
- 清理链接
- 复读机
- 消息统计
- 按照条件解除频道消息在群组的置顶

# 运行
安装好依赖之后
```bash
uv run main.py
```
